# **App Name**: IDC Ops Center

## Core Features:

- Server Selection: Browse and select servers from a searchable, sortable list, filterable by attributes visible in the original screenshot.
- Operation Selection: Choose the type of operation to perform on the selected server(s) from options available in the original screenshot.
- Parameter Input: Provide necessary parameters for the selected operation (e.g., script parameters, reboot options, etc.).
- Scheduled Execution: Schedule the operation to be executed at a specific time, if needed.
- Confirmation and Submission: Review all details of the operation before final submission.
- Operation Summary: Receive and display a clear summary of actions

## Style Guidelines:

- Primary color: A deep, authoritative blue (#336699), evoking trust and reliability.
- Background color: A light, neutral gray (#F0F0F0) to ensure readability and reduce eye strain during long work sessions.
- Accent color: A vibrant, complementary orange (#FF9933) to draw attention to key interactive elements.
- Body and headline font: 'Inter' sans-serif for a modern and clear appearance; suitable for both headlines and body text.
- Use a consistent set of simple, line-based icons to represent different operations and server statuses.
- Adopt a single-column layout to ensure ease of use on smaller screens. Use expandable sections and accordions to organize different functional areas.
- Subtle transitions when expanding/collapsing sections, confirming actions, and submitting operations to give a modern feel
